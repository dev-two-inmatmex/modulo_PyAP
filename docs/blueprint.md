# **App Name**: Hola World

## Core Features:

- Simple Display: Display the text 'Hola Mundo' on the screen.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey a sense of calmness and reliability.
- Background color: Light gray (#E0E0E0), a desaturated version of the primary, for a neutral backdrop.
- Accent color: Violet (#9C27B0), an analogous hue to the primary color, for interactive elements.
- Font: 'Inter' (sans-serif) for clear, readable text.
- Center the 'Hola Mundo' text on the screen.
